# 🎵 Background Music

Place your romantic background music MP3 file here.

**Default expected filename:** `romantic-background.mp3`

## Where to get free romantic music:

1. **Free Music Archive** — https://freemusicarchive.org
2. **Pixabay Music** — https://pixabay.com/music/ (search "romantic piano")
3. **Bensound** — https://www.bensound.com (search "romantic")

## Steps:
1. Download a soft piano / romantic background track (.mp3)
2. Rename it to `romantic-background.mp3`
3. Place it in this folder: `public/music/`

## Change the file name:
Edit `src/data/config.js` → update `CONFIG.musicFile` to match your filename.
